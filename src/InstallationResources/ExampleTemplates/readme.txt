= About the example =
NOTE! These files are part of the installation package for EPiServer CMS Manager. You will only need these if you do a manual installation.

In the `newsletter` directory is the example templates for enabling the newsletter functionality on your site. Please make sure your newsletter directory is directly below your site root as the page types is configured to use that path.

The url should look something like this: 
`http://www.mysite.com/newsletter/newsletter.aspx`
